SQL Assignment – Module 5 Final Project
📌 Project Overview
This project is part of the SQL for Data Science coursework. It demonstrates practical applications of SQL queries using SQLite to analyze and manipulate datasets. The notebook (mod5_final_project.ipynb) contains step-by-step solutions to the final assignment, showcasing skills in database querying, filtering, aggregation, and joins.

🎯 Learning Objectives
Understand relational database concepts.

Write SQL queries to:

Select and filter data.

Perform aggregations with GROUP BY and HAVING.

Use JOIN operations to combine tables.

Apply subqueries and nested queries.

Interpret query results for data-driven insights.
